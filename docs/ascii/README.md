# Game Processes (ASCII + PNG)

Этот каталог содержит end-to-end диаграммы процессов игры по основным use-case.

Форматы:
- `*.puml` — исходники PlantUML
- `*.utxt` — ASCII/Unicode версия диаграммы
- `*.png` — картинка для быстрого просмотра

Список диаграмм:
- `uc01_player_session_and_profile_polling`
- `uc02_admin_panel_polling_and_actions`
- `uc03_backend_game_loop_and_battle_tick`
- `uc04_live_leaderboard_public_polling`
- `uc05_game_lifecycle_states`
- `uc06_system_context_actors`
- `uc07_build_leaderboard_who_when_why`
